# SafeHaven

> A safe, calm, and joyful learning space built for children with autism.

SafeHaven is a Java desktop application designed to make early learning
a positive and stress-free experience. Children can explore letters, numbers,
and sounds at their own pace — through soft visuals, gentle interactions,
and fun activities that never feel overwhelming.

Parents stay connected through a dedicated dashboard that tracks every
milestone their child reaches.

---

## Designed With Autistic Children in Mind

Every feature in SafeHaven was built around one goal — to make a child
feel safe, confident, and happy while learning. No timers. No pressure.
No sensory overload. Just calm, guided discovery.

---

## Features

### Letters & Numbers
- Learn the A to Z alphabet and 0 to 9 numbers through clear, colorful visuals
- Each letter and number is introduced gently — one at a time —
  so the child never feels rushed or overwhelmed

### Sounds & Audio Learning
- Hear the sounds of every letter and number
- Explore sounds of animals — cat, dog and cow
- Discover sounds of musical instruments — tabla, flute and bell
- Audio-based learning helps children build recognition and memory
  in a fun, engaging way

### Bubble Bursting Game
- A calm and satisfying mini-game where children pop bubbles
- Helps with focus, hand-eye coordination, and sensory engagement
- Gentle and repetitive — great for children who enjoy soothing activities

### Shape Matching Game
- Match shapes to their correct outlines
- Builds visual recognition, problem solving, and concentration
- Simple enough to feel achievable, engaging enough to keep playing

### Pixel Art Drawing
- A free, creative pixel canvas where children can draw and color freely
- No rules, no wrong answers — just pure creative expression
- Encourages fine motor skills and self-expression in a pressure-free space

## Parent Dashboard

Parents have their own separate and secure login — completely
independent from the child's experience.

### How it works
1. Parent signs up with their email and password
2. They add their child's profile using the email and password
   the child used when signing up
3. The dashboard then displays the child's full activity history

### What parents can see
- Activities completed and time spent on each
- Scores and progress in games and learning modules
- Overall learning milestones reached over time
- A clear picture of where their child is thriving and
  where they may need more practice

> Parents stay involved in their child's learning journey without
> ever interrupting it.

## How to Run

1. Clone the repository:
```bash
   git clone https://github.com/Sobia-khalid770/SafeHaven.git
```

2. Open Eclipse and go to:
   **File → Import → General → Existing Projects into Workspace**

3. Select the cloned folder and click **Finish**

4. Run `Main.java` inside the `src` folder

No extra installations needed. Just Java and Eclipse.

## Motivation

This project was built as part of my academic studies with a deeper
purpose in mind — to create something that genuinely helps.

Children with autism often face challenges with overstimulation,
traditional learning formats, and environments that don't adapt to
their needs. SafeHaven was designed to be different — a place where
learning happens gently, creatively, and on the child's own terms.

The parent dashboard bridges the gap between the child's progress
and the parent's awareness — keeping families connected in the
learning journey without pressure or interruption.
